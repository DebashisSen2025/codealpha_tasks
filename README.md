# 📊 Exploratory Data Analysis (EDA) in Banking Using Python

CodeAlpha Data Analytics Internship — Task 2: Exploratory Data Analysis

---

# 🎯 Project Overview

This project performs a complete **Exploratory Data Analysis (EDA)** on the **UCI Bank Marketing Dataset** using Python and Pandas.

The objective of this analysis is to understand **customer behavior** and identify patterns that can help banks design **more effective marketing campaigns for term deposits**.

# 📁 Dataset

Source: UCI Machine Learning Repository — Bank Marketing Dataset  
Reference: Moro et al., 2014  

Dataset Details:

Total Records: 41,188  
Total Features: 21  

Target Variable:  
**y** — Whether the client subscribed to a term deposit (yes/no)

# ❓ Business Questions Answered

1. What percentage of clients subscribed to the term deposit?
2. What are the average values of numerical features among subscribed clients?
3. What is the average call duration for subscribed clients?
4. What is the average age among unmarried subscribed clients?
5. How do age and call duration vary across different job types?
6. Which months produce the highest campaign conversion rates?
7. Which job types and education levels convert the most?

# 🔑 Key Insights

1️⃣ Only **11.3% of clients subscribed**  
→ Marketing campaigns require better targeting.

2️⃣ **Students and Retired clients convert the most**  
→ These groups should be prioritized in campaigns.

3️⃣ **Call duration strongly impacts conversion**  
→ Training call agents to improve engagement can increase success.

4️⃣ Clients **under 30 and over 60 convert more**  
→ Age-targeted marketing strategies may improve results.

5️⃣ **March, September, October, and December perform best**  
→ Major campaigns should be scheduled during these months.

6️⃣ **University-educated clients convert more**  
→ Digital marketing may work better for this segment.

7️⃣ High **euribor3m interest rates reduce subscriptions**  
→ Economic indicators should be considered when planning campaigns.
